<ul>
        <li>파일 확장자 : .bs</li>
        <li>메인 헤더 : ShootPoint 개수, ShootPoint 위치(파일 내부에서)</li>
        <li>ShootPoint 헤더 : ShootPoint 좌표, Bullets 개수, 각 Bullet 위치(파일 내부에서)</li>
        <li>Bullet : 시작시간(s), 방향(도), 크기(반지름), 속도, 형태, 색, 반복회수, 반복간격, 각속도, 가속도</li>
</ul>
